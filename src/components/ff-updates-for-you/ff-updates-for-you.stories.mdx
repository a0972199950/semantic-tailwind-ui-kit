### Properties

| Property     | Attribute | Description | Type          | Default | Requirement
| ------------ | --------- | ----------- | ------------- | ------- |
| `benchmarks` | --        | A set of data that you want to display in component     | `Benchmark[]` | `[]`    | `false`

- Benchmark

Key | Type | Description
--|:--|:--
key | string | The unique key with this data
Text | string | String that will displayed on the component
Score | number | 0~10, each range of scores represent different color

### Events

| Event            | Description | When to trigger               |
| ---------------- | ----------- | ------------------ |
| `detailBtnClick` |             | When user click the button "See details" |
