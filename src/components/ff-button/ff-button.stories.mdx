### Properties

| Property     | Attribute | Description | Type          | Default | Requirement
| ------------ | --------- | ----------- | ------------- | ------- |
| `type` | --        | Button color | `string` | `primary`    | `false`
| `size` | --        | Button size | `string` | `medium`    | `false`

### Events

| Event            | Description | When to trigger               |
| ---------------- | ----------- | ------------------ |
| `click` |             | When user click the button |
